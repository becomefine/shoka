# 初识Vue.js

## 1.1、Vue.js是什么?
Vue.js是简单小巧的核心，渐进式技术栈，提供了现代Web开发中常见的高级功能，
- 解耦试图与数据
- 可复用的组件
- 前端路由
- 状态管理
- 虚拟DOM（virtual DOM）。

## 1.2 如何使用Vue.js

```html
<script src = "https://unpkg.com/vue/dist/vue.min.js"></script>
<!--指定某个版本的Vue.js-->
<script src = "https://unpkg.com/vue@2.1.6/dist/vue.min.js"></script>
```
